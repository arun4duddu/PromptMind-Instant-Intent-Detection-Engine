# PromptMind

## https://open-vsx.org/extension/promptmind/promptmind

## Better prompts → Better Copilot responses → Faster development

AI-powered VS Code extension that classifies natural language prompts and auto-generates optimized Copilot Chat prompts using proven prompt-engineering best practices.

It helps developers, engineers, and product builders write **better prompts faster** — so you get higher-quality responses from GitHub Copilot.

## ✨ What It Does

- You write a simple natural language request  
- The extension intelligently classifies the intent  
- You choose a **Prompt Type** from the UI  
- The extension generates **4 optimized prompt variations**

You can then:

- 📋 Copy any prompt  
- 🚀 Send directly to Copilot Chat  

This removes guesswork from prompt writing and improves Copilot output quality.

---

## 🎯 Why Use This?

Most Copilot users waste time rewriting prompts or struggle with:

- Vague outputs  
- Poor structure  
- Missing context  
- Inconsistent quality  

PromptMind solves this by applying proven prompt-engineering frameworks automatically.

## 🖥️ How It Works

1. Open the extension inside VS Code  
2. Enter your raw prompt (e.g., *"Write unit tests for this service"*)  
3. Select a prompt type (e.g., **Test Case Prompt**)  
4. Instantly receive 4 optimized prompt options  
5. Copy or send directly to Copilot Chat
